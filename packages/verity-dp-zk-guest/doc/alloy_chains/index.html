§alloy-chains

Canonical type definitions for EIP-155 chains.

§Supported Rust Versions

Alloy will keep a rolling MSRV (minimum supported rust version) policy of at least 6 months. When increasing the MSRV, the new Rust version must have been released at least six months ago. The current MSRV is 1.81.0.

Note that the MSRV is not increased automatically, and only as part of a minor release.

§Adding a new chain

Check src/named.rs’s comment for the guidelines on how to add a new chain.

§Note on no_std

All crates in this workspace should support no_std environments, with the alloc crate. If you find a crate that does not support no_std, please open an issue.
