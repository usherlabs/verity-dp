Expand description

This trait is used to encode values using a global offset (delta).

Implementations of this trait should be idempotent, meaning that calling encode multiple times with the same id should return the same result. This way every value is assigned a unique encoding which can be regenerated at any time, independent of the order in which values are encoded.
