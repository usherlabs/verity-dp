Verity Data Processor (VDP) Framework

Packages

  • verity-ic-0.2.5-rc.4 Utilities for Verity IC (Internet Computer) components, used by Managed Verifier or third-party IC Canisters.
  • verity-verify-local-0.2.5-rc.4 Utility for locally verifying signed Merkle tree responses obtained from the managed Verity Verifier
  • verity-verify-remote-0.2.5-rc.4 Utility for delegating TLS proof verification to managed Verity Verifier, returning a signed Merkle tree for local/on-chain/in-zkVM verification.
  • verity-verify-tls-0.2.5-rc.4 Utility for verifying Verity TLS proofs
  • verity-client-0.2.5-rc.4 Verity Rust Client
  • verity-dp-zk-guest-0.2.5-rc.4 Utilities for Verity zkTLS components that reside in the guest/zkVM
  • verity-dp-zk-host-0.2.5-rc.4 Utilities for Verity zkTLS components that reside in the host, orchestrating the zkVM to aggregate TLS proofs.
